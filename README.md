# Dynamic Student Portfolio Builder

This project is a responsive student portfolio website created using HTML and CSS.  
It allows students to showcase their personal information, skills, and projects in a clean and professional layout.

---

## Features

- Responsive portfolio layout
- Personal profile section
- About / Bio section
- Skills display section
- Projects showcase section
- Clean lavender and black theme
- Easy to customize

---

## Technologies Used

- HTML
- CSS

---

## How to Run

1. Download or clone the repository
2. Open the HTML file in any browser (Chrome, Edge, Firefox)
3. Edit the content to add your personal details
